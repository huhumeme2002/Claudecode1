Hướng dẫn cài đặt Claude Code

Kết nối Claude Code với API proxy chỉ với vài bước đơn giản.

Model hỗ trợ: claude-opus-4-6, claude-sonnet-4-6, claude-haiku-4-5-20251001

1 Tải các thư viện cần thiết

Cài đặt Node.js — Tải tại nodejs.org (chọn bản v22.x LTS)

node -v

Cài đặt Bun

macOS / Linux:

curl -fsSL https://bun.sh/install | bash

Windows:

powershell -c "irm bun.sh/install.ps1 | iex"

2 Chạy lệnh cài đặt

Mở terminal và chạy lệnh sau:

bunx prox-claude-installer@latest

Installer sẽ tự động cấu hình Claude Code với API proxy.

3 Sử dụng API trực tiếp

Tương thích hoàn toàn với Anthropic SDK. Chỉ cần thay base_url trỏ về http://pro-x.io.vn

Python

import anthropic client = anthropic.Anthropic( api_key="sk-your-api-key", base_url="http://pro-x.io.vn" ) message = client.messages.create( model="claude-sonnet-4-6", max_tokens=1024, messages=[ {"role": "user", "content": "Xin chào!"} ] )

Node.js

import Anthropic from "@anthropic-ai/sdk"; const client = new Anthropic({ apiKey: "sk-your-api-key", baseURL: "http://pro-x.io.vn", }); const message = await client.messages.create({ model: "claude-sonnet-4-6", max_tokens: 1024, messages: [ { role: "user", content: "Xin chào!" } ], });

cURL

curl http://pro-x.io.vn/v1/messages \ -H "Content-Type: application/json" \ -H "x-api-key: sk-your-api-key" \ -H "anthropic-version: 2023-06-01" \ -d '{ "model": "claude-sonnet-4-6", "max_tokens": 1024, "messages": [ {"role": "user", "content": "Xin chào!"} ] }'

Lưu ý quan trọng

  • Base URL: http://pro-x.io.vn — KHÔNG thêm /v1 khi dùng SDK
  • Khi dùng cURL, endpoint đầy đủ là http://pro-x.io.vn/v1/messages
  • API key có thể dùng qua header x-api-key hoặc Authorization: Bearer

Hướng dẫn cài đặt OpenClaw

Kết nối OpenClaw với API proxy để sử dụng Claude trong terminal.

Model hỗ trợ: claude-opus-4-6, claude-sonnet-4-6, claude-haiku-4-5-20251001

1 Cài đặt OpenClaw

Yêu cầu: Node.js v22.12+ — Tải tại nodejs.org (chọn bản v22.x LTS)

node -v

Cài OpenClaw:

npm install -g openclaw@latest

2 Chạy Onboard

openclaw onboard --install-daemon

4 Sửa Context Window (Quan trọng!)

Sau khi onboard xong, bắt buộc phải sửa 2 file:

File 1: models.json

Windows: %USERPROFILE%\.openclaw\agents\main\agent\models.json

macOS/Linux: ~/.openclaw/agents/main/agent/models.json

File 2: openclaw.json

Windows: %USERPROFILE%\.openclaw\openclaw.json

macOS/Linux: ~/.openclaw/openclaw.json

Tìm và sửa thành:

"contextWindow": 200000, "maxTokens": 8192
⚠ Phải sửa cả 2 file. Mặc định wizard để 16000/4096 — quá thấp, sẽ gây lỗi compaction loop.

5 Khởi động OpenClaw

openclaw gateway

Nếu terminal hiển thị gateway khởi động thành công, bạn đã sẵn sàng sử dụng!

Nếu gặp lỗi, hãy liên hệ admin để được hỗ trợ.

Gia hạn / Nâng cấp gói

Chọn gói bạn muốn mua. Sau khi thanh toán, bạn sẽ nhận được API key mới với gói tương ứng.
